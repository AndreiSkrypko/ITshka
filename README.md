# ITshka — Создание сайтов в Лиде

Веб-студия полного цикла в Лиде. Создание сайтов под ключ, разработка лендингов, интернет-магазинов, корпоративных сайтов.

## Технологии

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- i18next (русский, польский, английский)

## Установка и запуск

```sh
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Просмотр production сборки
npm run preview
```
